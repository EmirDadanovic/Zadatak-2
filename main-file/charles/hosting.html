DETALJI NASIH SULUGA

HOSTING

Neprekidno radimo od 1998. godine i Vodeći smo hosting provajder u BiH. Svojim klijentima nudimo uslugu vrhunskog kvaliteta, profesionalnu 24h podršku i trostruki back up svih podataka (dnevni, sedmični i mjesečni). Leftor hosting paketi idealni su za svakoga ko svoje stranice i e-mailove želi staviti na kvalitetan i pouzdan server.

  • Web win Hosting

    Win Web Hosting Vam je potreban ako WEB sajt pravite u nekoj od Microsoft tehnolgija kao što je: ASP, .NET, ako koristite C# ili VB.NET programski jezik. Takođe, ako Vam treba podrška za SQL server to možete dobiti u našim WIN Web hosting paketima.

  • Specijalizirani hosting

    Specijalizirana hosting usluga je posebno prilagođena potrebama portala da u toku Facebook kampanje prime veliki promet sa Facebook Fan stranica. Ovi serveri mogu da obrade i preko 3500 posjetilaca u realnom vremenu, čak i ako svi ti posjetioci dođu u iste 2-3 minute na web stranicu

  • Reseller gosting

    Vlasnik Reseller paketa ima mogućnost da kroz cPanel / WHM WEB sučelje doda proizvoljan broj stranica (domena) kojima dodjeljuje dio resursa kojima raspolaže. Vlasnik takođe može po potrebi da radi supenziju i unuspenziju neplaćenih paketa, da centralizirano šalje obavještenja klijentima, da radi terminaciju ili prepaketisanje.

Virtualni privatni serveri

Trebate hosting rješenje sa potpunom kontrolom nad serverskim resursima, a ne želite ulagati u vlastiti server. Virtuelni server je idealno rješenje za Vas. Leftor Vam nudi tri predefinisana paketa VPS servera, ali ako smatrate da nijedan od naših standardnih VPS hosting paketa ne bi zadovoljio Vaše potrebe, nudimo Vam i uslugu VPS po mjeri koja Vam omogućava da sami složite savršeni VPS server prema svojim potrebama. Na svaki VPS server je moguće podesiti operativni sistem po želji, a posebno pripremljene “image” imamo za Ubuntu Server i Desktop verziju i za CentOS.
